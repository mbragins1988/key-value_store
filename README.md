# KVStore — gRPC key-value store

In-memory key-value хранилище с поддержбой TTL и LRU eviction. Реализовано на Python с gRPC

## Зачем

Хранилище ключ-значение в памяти с ограничением 10 ключей. Поддерживает:
- TTL (время жизни ключа)
- LRU вытеснение (удаляется наименее используемый ключ при превышении лимита)
- Параллельные запросы

## Запуск

```bash
# Запуск сервера
uv run python server.py

# В другом терминале — клиент
uv run python client.py
